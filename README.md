### Introduction
NexusDroid is a WebView based store for applications and games. It lists both official **and** modified versions of what you're looking for. It's lightweight, fast and simple. There are many official and unofficial stores that currently exist but most have one thing in common. You can only find one genre of application or game (official / modified). The goal of NexusDroid is to provide you with both options, in one place.

---

### How It Works
The APK was built using Niotron (a platform that utilizes WebView for creating mobile applications). The WebView displays the contents of this repository. As for the actual files, they are downloaded from several sources and then listed under respective categories. Below, is a list of sources that we use for the files:

### Usage
Simply download the APK file from [here](https://google.com/404) or from the releases section. The APK doesn't require any updates as it's a WebView and is just displaying contents of this repository. However, if you notice that updates aren't visible to you, follow the steps below:

1. Force stop / kill the application (hold down on the application icon and select "Kill" or do so via developer options)
2. Clear the application cache and data via settings
3. Wait 5 - 10 seconds and then re-open the application

This will fix the issue (if you should come across it).

### Notes & Information
- Some may be wondering why we're using WebView instead of building an actual application. While we could have done this, we chose a more lightweight option that's simple to use, uses barely any resources and doesn't require any updates. Of course it's easier for us to develop but it's also convenient for the user.
- For those worrying if NexusDroid will work on earlier Android versions, you can rest easy. As long as you're running Android 10 or above, you're all good.
